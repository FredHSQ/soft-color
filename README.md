## Inicializating project

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/FredHSQ/soft-color-game/
   ```
3. Go to the project
   ```sh
   cd .\soft-color-game\
   ```
4. Install all packages
   ```sh
   npm install;
   ```
   or
   ```sh
   yarn install;
   ```
5. Start project
   ```sh
   npm run dev;
   ```
6. Test project
   ```sh
   npm test;
   ```
